# Imperium 产品文档索引

- 阶段：产品定义
- 范围：仅讨论产品层，不进入实现细节
- 语言：中文

## 使用说明

本目录承载 `Imperium` 的正式产品文档。

文档拆分原则：

- 每个文档只解决一类问题
- 优先保证评审可读性，而不是一次性堆满所有内容
- 当前阶段只讨论“产品要做什么”，不讨论“代码怎么写”

## 文档列表

### `01-产品定位与原则.md`

回答问题：

- 这是什么产品
- 解决什么问题
- 为什么要这样设计
- 产品的原则和边界是什么

### `02-组织角色与权责.md`

回答问题：

- 恺撒体系里有哪些角色
- 每个角色的职责和边界是什么
- 这些角色为什么存在

### `03-运行模式与状态机.md`

回答问题：

- 产品支持哪几种运行模式
- 一个议案如何流转
- 哪些状态是关键节点

### `04-权限矩阵与核心流程.md`

回答问题：

- 谁能调用谁
- 谁能否决谁
- 关键流程怎样闭环
- 哪些制度规则不能破坏

### `05-界面与数据模型.md`

回答问题：

- 产品的主要页面是什么
- 核心对象长什么样
- 活动流和案卷如何建模

### `06-调度恢复与交付范围.md`

回答问题：

- 调度和恢复怎么设计
- MVP 做到哪里为止
- 后续路线图是什么
- 当前阶段如何验收

## 建议阅读顺序

1. 先读 `01-产品定位与原则.md`
2. 再读 `02-组织角色与权责.md`
3. 接着读 `03-运行模式与状态机.md`
4. 然后读 `04-权限矩阵与核心流程.md`
5. 再看 `05-界面与数据模型.md`
6. 最后看 `06-调度恢复与交付范围.md`

## 当前阶段边界

当前阶段不做：

- 技术选型定稿
- 接口实现
- Prompt 实现
- 数据库表结构
- 前端视觉细化

当前阶段产出将作为后续设计阶段的输入。
