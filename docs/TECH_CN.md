## 文档说明

本文件作为 `Imperium` 技术方案总入口，不承载所有技术细节正文。

技术文档采用分层结构，目的如下：

- 避免单个技术文档过长，不利于评审
- 将总体架构、领域模型、数据设计、Agent 集成、接口设计、部署方案分层沉淀
- 让后续研发、测试、部署工作有清晰输入

## 当前技术基线

本轮技术方案已明确采用以下基线：

- 后端：Java 21 + Spring Boot 3.x
- 数据库：MySQL 8.4 LTS
- 消息队列：RocketMQ 5.x
- Agent 运行方式：OpenClaw CLI 调用
- 前端：全新页面实现
- 部署：Docker / Docker Compose
